# Fusabi Community vNEXT Documentation

This directory contains documentation for the next release of fusabi-community.

## Overview

Fusabi Community is a collection of community-maintained packages, libraries, and tools for the Fusabi programming language. This repository serves as a registry and distribution point for high-quality, reusable Fusabi code.

## Quick Start

1. Browse available packages in the [registry](../../../registry/index.toml)
2. Check individual package documentation in `packages/<package-name>/README.md`
3. Review examples in `packages/<package-name>/examples/`

## Package Categories

- **Terminal UI**: TUI components and applications
- **Observability**: Logging, tracing, and monitoring utilities
- **K8s/Cloud**: Cloud infrastructure and Kubernetes integrations
- **MCP/AI**: Model Context Protocol and AI integrations
- **Core Utilities**: JSON parsers, file managers, and other utilities
